## 📌 구현 기능
#### ➡️  반응형웹
다양한 디바이스 환경에서 적용되는 랜딩페이지를 디자인대로 구현
- 휴대폰 사진은 모바일로 축소시 사진처럼 글자 밑에 위치해야함
- 배경은 background에 선형 그라디언트로 효과 줄것
- 버튼 hover일때 효과

#### ➡️  애니메이션
애니메이션 변화를 부드럽게 적용하고, 디자이너가 원하는 세세한 표현 구현
- 이미지 hover일때 부드럽게 하얀 border 생기도록
- 애니메이션 동작시 이미지 크기가 변하지 않아야함
